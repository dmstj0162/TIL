# ES6 정리

1. 클래스

- ES6에서 도입된 객체를 생성하기 위한 새로운 설계도 양식이다.
- 기존의 생성자 함수보다 훨씬 더 명확하고 체계적으로 객체를 설계할 수 있다.

<pre>
<code>
class Student{
    constructor(name, group){
        this.name = name;
        this.group = group;
    }

    introduce(){
        console.log(`안녕하세요 ${this.group}반 ${this.name}입니다.`)
    }
}

const studentA = new Student('홍길동', 'A')
const studentB = new Student('유관순', 'B')

// 모든 인스턴스가 같은 함수를 쓴다.
console.log(studentA.introduce === studentB.introduce);
</code</pre>

- 기본 변환

- 일반 함수의 this는 호출할 때 결정된다.
- 즉, 누가 부르냐에 따라 바뀜
- 하지만 화살표 함수는 자신만의 this를 가지지 않고, 바깥 스코프의 this를 그대로 쓴다.
- 객체 메서드 = 무조건 일반 함수 사용 / 화살표 함수 사용시 this 깨

4. Reset parameter, spread syntax

- Reset Parameter: 여러 개의 값을 하나의 배열로 모으기
- Spread Syntax: 하나의 배열을 여러 개의 값으로 펼치기

- Rest 파라미터는 반드시 매개변수 목록의 가장 마지막에 위치해야 한다.
<pre>
    <code>
      function merge(first, ...args){
        console.log(`첫 번째: ${first}`);
  console.log("나머지(배열):", args);

  let message = first;
  for (let arg of args) {
    message += arg;
  }
  return message;
      }
