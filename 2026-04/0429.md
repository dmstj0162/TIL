## 변수 선언

- 자료형 변수명;
- 자료형이란?
- 다양한 값의 형태별로 어느정도 크기를 하나의 값으로 취급할 것인지 미리 Compiler와 약속한 키워드
- 기본자료형은 8가지가 있다.

<pre><code>
/*
        * 정수를 취급하는 자료형
        * */

        System.out.println("multi = " + multi); // soutv

## 자동형변환

<pre><code>
    /*
        * 자동 형변환
        *
        * 자바에서 다른 타입끼리의 연산은 피연산자들이 모두 같은 타입인 경우 실행할 수 있다.
        * 즉, 같은 데이터 타입끼리만 연산을 수행할 수 있다.
        * */

        // 1. 작은 자료형에서 큰 자료형으로는 자동 형변환이 된다.
        byte bnum = 1;
        short snum = bnum;
        int inum = snum;
        long lnum = inum;

        // 연산 시에도 자동으로 큰 자료형에 맞춰 계산
        int num1 = 10;
        long num2 = 10;

//        int result1 = num1 + num2; // long으로 변경 후 계산 -> int 변수에 담을 수 없음
        long result1 = num1 + num2;

        /*2. 정수는 실수로 자동 형변환 된다.*/
        long eight = 8;
        float four = eight;

        float result2 = eight + four;

        /*3. 논리형은 형변환 규칙에서 제외
        * 어느 자료형이든 boolean을 형변환 해서 담을 수 없음*/
        boolean isTrue = true;
//        byte b = isTrue;
//        short s = isTrue; ... 등

        /*강제형변환
        * 바꾸려는 자료형으로 캐스트 연산자를 이용하여 형변환한다.
        * (바꿀자료형) 값;
        * */

        long lnum1 = 8;
//        int inum1 = lnum1; // 에러
        int inum1 = (int)lnum1; // 변경하려는 자료형을 명시하여 강제형변환

        short snum1 = (short)inum1;

        /*실수를 정수로 변경 시 강제 형변환이 필요*/

        float fnum2 = 4.0f;
        long lnum3 = (long)fnum2;     // 데이터 손실 감안할테니 형변환 해줘

</code></pre>
