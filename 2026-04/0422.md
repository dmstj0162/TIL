- Next.js 특징

1. 서버 사이드 렌더링(SSR) 지원
2. 파일 기반 라우팅
3. 정적 사이트 생성(SSG)과 하이브리드 렌더링
4. 풀 스택 개발 환경

- usePathName() : 현재 url 경로를 읽어옴 -> 활성 탭/메뉴 스타일 적용시
- useParams() : [폴더명] 형태의 동적 라우팅 파라미터를 읽어옴 -> 상세페이지(/menu/1)
- useSearchParams(): url 끝의 쿼리스트링(?key=value)을 읽어옴 -> 검색 결과 페이지, 필터링
- useRouter(): 함수/이벤트 내에서 프로그래밍 방식으로 페이지 이동 -> 폼 제출 완료 페이지 이동, 뒤로가기

** 'use client' 클라이언트 컴포넌트에서만 사용 가능 **
