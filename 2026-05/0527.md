## @Component

- Spring이 관리할 일반 Bean
- @Controller = 웹 요청 처리 계층 bean
- @Service = 비즈니스 로직 계층 bean
- @Repository = 데이터 접근 계층 bean
- @Autowired = 필요한 의존 객체를 자동으로 주입

## 주입방식

1. 필드 주입
2. setter 주입
3. 생성자 주입

## @Primary

- 기본 우선 Bean 지정

## @Quarlifier

- 이름으로 특정 Bean 지정

## List<T>

- 같은 타입 Bean 여러 개를 한 번에 주입

## Bean Scope

- singleton : Container 안에서 하나만 만들어지고 공유(Service, Repository)
- prototpe : getBean() 호출 때마다 새로 만들어짐

## Bean Life Cycle

- Bean은 생성, 의존성 주입, 초기화, 사용, 소멸 흐름을 가진다
- initMethod, destroyMethod로 특정 시점에 메소드를 실행할 수 있다.
- @PostConstruct, @PreDeatroy 방식도 있다(참고)

## 외부 설정값

- 변경 가능성이 있는 값은 Java 코드에 직접 박아두지 않는 것이 좋다.
- properties 파일에 값을 작성하고 @PropertySource, @Value로 읽을 수 있다.
