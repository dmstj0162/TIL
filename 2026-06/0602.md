## Filter

-Servlet 스펙
-Spring MVC에 들어오기 전 요청
-DispatcherServlet 앞단에서 동작

## Interceptor

-Spring MVC 기능
-Controller 실행 전 후 요청 처리
-DS이 Controller 찾은 뒤, Controller 호출 전 후에 동작

## AOP

-메서드 실행 전 후 공통 로직 처리
-Service, Repository 등 다양한 Bean 메서드에 적용 가능
-HTTP 요청이 아니어도 Spring Bean 메서드 실행 흐름에 적용 가능

## 200 OK

- 조회 성공
- 응답 body 있음

## 201 Created

- 새 리소스 생성 성공
- Location header로 새 리소스 위치 안내 가능

## 204 No Content

- 요청 처리 성공
- 응답 body 없음
- 수정/삭제 성공에 자주 사용

## ResponseMessage

- 응답 body 구조를 일정하게 맞추기 위한 DTO
