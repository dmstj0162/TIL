## Dynamic SQL

- 조건에 따라 SQL을 바꿀 수 있음

Mapper(타입 안정성) + XML(SQL 분리 + 동적 쿼리 가독성)

gte = greater than

gte >=
lte <=
gt >
lt <

- MyBatis에서는 Mapper 인터페이스 + XML을 이용해서 SQL을 분리하고, JDBC의 반복 작업(Connection, PreparedStatement, ResultSet 처리)을 자동화하며, <if>, <where>, <foreach> 같은 Dynamic SQL 기능으로 조건에 따라 SQL을 자동 생성하는 방법

1. Mapper 인터페이스 = DAO 역할
2. Mapper XML = SQL 작성하는 곳
3. #{} = JDBC의 ?
4. resultMap = 컬럼명과 DTO 변수명 연결
5. <if>, <where>, <foreach> = Dynamic SQL
