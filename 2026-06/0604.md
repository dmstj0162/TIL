## MyBatis

- 데이터의 입력, 조회, 수정, 삭제(CRUD)를 보다 편하게 할 수 있도록 xml로 구조화한 Mapper 설정 파일을 통해 JDBC를 구현한 영속성 프레임워크
- 기존의 JDBC를 통해 구현했던 상당 부분의 코드, 파라미터 설정 및 결과 mapping을 xml 설정으로 쉽게 구현할 수 있게 한다.
- JDBC의 반복 코드를 없애주는 도구
- SQL을 실행하며, ResultMap으로 DB 컬럼명과 DTO 필드명을 연결하고, #{필드명}은 DTO의 getter를 호출해 값을 가져온다.

- sqlSession : sqlSession Factory 필요

- Mapper : SQL 실행기
- #{name}은 DTO의 getName() 호출

- #{} 안에는 DB 컬럼명이 아닌 DTO의 property 명을 적어야 한다.
